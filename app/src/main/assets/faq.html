Tips

  • You can obtain extra information about a suggested method by a long press

Limitations:

  • CGO is not available to because it relies on gcc. I may support it in the future by using an Android port of gcc.
  • Same goes for Go Get since it requires multiple version control systems to be already installed on the machine.
  • Everything is stored in internal memory for convenience because binaries can't run on the SD card.
  • Manipulating the I/O streams directly does not work because current Android versions do not support I/O redirection.
